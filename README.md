## About

Merhaba, ben Abdulkadir. Pratik araçlar, otomasyon scriptleri, Chrome eklentileri ve FiveM/QBCore ekosistemi için güvenlik ve yönetim odaklı sistemler geliştiriyorum.

Kod yazarken odaklandığım şeyler:

- hızlı kullanılabilen küçük ama net araçlar
- oyun sunucuları için güvenlik, kontrol ve yönetim scriptleri
- browser automation ve extension tabanlı çözümler
- Python, JavaScript/TypeScript, Lua ve PHP ile ürünleştirilebilir yardımcı sistemler
